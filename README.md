# Piper-Chat
Real Time Private Chat app with specific allowed users only. It is mobile responsive. 

## Made with
 Node.js <br />
 Express.js <br />
 MongoDB Atlas <br /> 
 React.js <br />
 Socket.io <br />
 Redux <br />
 Mocha and chai
<br />
<br />

## Installation 
```
npm install
cd client 
npm install 
cd ..
npm run dev
```
