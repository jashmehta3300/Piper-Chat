# Piper-Chat
### Real Time Private Chat app with specific allowed users only.

## Made with
 Node.js <br />
 Express.js <br />
 MongoDB Atlas React.js <br />
 Socket.io <br />
 Redux <br />
 Mocha and chai

## Intallation 
```
npm install
cd client 
npm install 
cd ..
npm run dev
```
